# VenueTwin Concierge

## 🚀 One-line Pitch

VenueTwin Concierge is a real-time AI digital twin for physical events that delivers personalized attendee journeys and predictive crowd intelligence.

---

## 🧠 What Problem Are We Solving?

Physical events are chaotic and inefficient:

* Attendees miss relevant sessions
* Popular sessions get overcrowded
* Networking is random and unstructured
* Organizers lack real-time visibility
* Sponsors cannot measure meaningful engagement

Current event apps are static — they show information but don’t adapt.

---

## 💡 Solution

VenueTwin Concierge creates a **live digital twin of the venue** and runs intelligent agents on top of it.

It actively:

* Guides attendees in real time
* Predicts crowd bottlenecks
* Recommends sessions dynamically
* Enables contextual networking
* Gives organizers a live control tower

This is not just an event app — it is a **live intelligence layer over a physical space**.

---

## ✨ Key Features

### 👤 Attendee Experience

* Personalized live itinerary
* Smart navigation with queue awareness
* Real-time session recommendations
* AI-powered networking matches

### 🏢 Organizer Control Tower

* Live venue heatmap
* Crowd congestion alerts
* Predictive bottleneck detection
* Session occupancy tracking

### 🤝 Networking Intelligence

* Match attendees based on interests
* Suggest contextual icebreakers
* Show proximity-based connections

---

## 🧩 Core Idea (What Makes This Unique)

Instead of static schedules, VenueTwin:

* Continuously adapts to real-time conditions
* Uses multiple intelligent agents
* Simulates a “living brain” over the venue

---

## 🏗️ Architecture Overview

Frontend: Next.js
Backend: FastAPI
Database: PostgreSQL + pgvector
Realtime: WebSockets
Cache/Event Bus: Redis
AI Layer: LLM-based agents + embeddings

---

## ⚙️ How It Works

1. Attendee checks in and selects interests
2. System builds a personalized itinerary
3. Live occupancy data updates venue state
4. Agents continuously adapt recommendations
5. Organizer dashboard shows crowd insights

## ▶️ How to Run Locally

```bash
# Clone repo
git clone https://github.com/your-username/venuetwin-concierge

# Backend
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

# Frontend
cd frontend
npm install
npm run dev
```
